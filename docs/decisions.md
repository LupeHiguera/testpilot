# Decisions

- Public project name: `testpilot`.
- Browser control: direct Playwright APIs for v1.
- Demo app: Vite React app in this repository.
- Model clients: deterministic mock mode plus optional OpenAI mode via `OPENAI_API_KEY`.
- Repair policy: auto-apply only validator-approved repairs to generated tests.
- Roadmap: add Playwright MCP and vision-assisted diagnosis as optional backends later.
